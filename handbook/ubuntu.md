# Ubuntu

### Permissions
- Type `sudo chmod a+rwx /path/to/file` into the terminal, replacing “`/path/to/file`” with the file you want to give permissions
- Type `sudo chmod -R a+rwx *` for all files in current folder

### Magic Wormhole
